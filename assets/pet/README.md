# Pet animation (Lottie)

Bỏ 4 file Lottie `.json` vào đây:

- `idle.json`  — pet đứng yên (mặc định)
- `focus.json` — lúc đang focus
- `break.json` — lúc nghỉ
- `done.json`  — lúc hoàn thành

Tải từ https://lottiefiles.com (chọn nút **Download → Lottie JSON**).

Chưa có file → app vẫn chạy, hiện emoji 🐱 thay thế.
